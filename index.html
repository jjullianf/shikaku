How to play

  1. The grid contains numbers — each one is a clue.
  2. Divide the entire grid into non-overlapping rectangles.
  3. Each rectangle must contain exactly one number.
  4. That number equals the area of its rectangle (width × height).
  5. Every cell must belong to exactly one rectangle.
